# Portfólio — Kayky Nery Alcântara Vieira

Portfólio pessoal estático desenvolvido com HTML, CSS e JavaScript puro.

## Estrutura

```
portfolio-kayky/
├── index.html       # Página principal
├── css/
│   └── style.css    # Estilos
├── js/
│   └── main.js      # Scripts (animações, scroll)
```

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, animações, media queries)
- JavaScript vanilla (IntersectionObserver)
